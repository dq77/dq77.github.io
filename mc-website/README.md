# mc-website 美仓官网仓库
- 效果预览地址： [https://www.diaoqii.com/mcweb/index.html](https://www.diaoqii.com/mcweb/index.html)
- 仓库地址： [https://github.com/dq77/mc-website](https://github.com/dq77/mc-website)

##运行方法
1. `npm install`
2. `npm run dev`

- 打包命令： `npm run build`
- 移动端适配可扫描二维码查看
